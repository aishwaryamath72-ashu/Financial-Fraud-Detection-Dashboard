# Financial Fraud Detection Dashboard

## Overview
This project uses Machine Learning to detect fraudulent financial transactions and visualize results using Streamlit.

## Technologies
- Python
- Pandas
- Scikit-Learn
- Streamlit
- Random Forest

## Features
- Fraud Detection
- Interactive Dashboard
- Transaction Analysis
- Fraud Statistics

## Run Project

Install packages:

pip install -r requirements.txt

Train model:

python model.py

Run dashboard:

python -m streamlit run app.py
